Collect Card Payments via Phone for Zen Cart

Version 1.0.0

Support Thread at Zen Cart Forums: https://www.zen-cart.com/showthread.php?230022-Collect-Card-Payments-by-Phone-Call-Support-Thread.

Zen Cart Plugin Download: https://www.zen-cart.com/downloads.php?do=file&id=2388.

What it does

This payment module enables your store to collect credit-card payments via phone call. You provide, as part of the payment module's configuration, the phone number you want customers to call. That's it, pretty simple.

The payment module provides a means for sites to transition from Manual Credit Card Collection payment modules (like Ceon Manual Card).

Version History:

  • v1.0.0, 2024-03-29 (lat9). Initial release.

Installation

The following files are provided. Simply copy them to your Zen Cart storefront, then navigate to your admin's Modules :: Payment tool to install and configure the payment module:

  1. /includes/languages/english/modules/payment/call_in_payment.php
  2. /includes/modules/payment/call_in_payment.php
